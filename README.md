# Create Toggle Button Animation in HTML CSS & JavaScript

Um projeto simples de **botão toggle animado (on/off)** usando HTML, CSS e JavaScript.

## ✨ Sobre

Este projeto demonstra como criar um **toggle button interativo com animação**, ideal para configurações e interfaces modernas.

## 🛠️ Tecnologias

- HTML
- CSS (animation, transition)
- JavaScript

## 🎯 Funcionalidades

- Botão toggle animado
- Alternância entre ligado/desligado
- Design moderno e interativo

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
